# Page Manifests

Each page manifest must conform to:
- `packages/ui-schemas/src/page-manifest.schema.json`

Path contract:
- `packages/ui-manifests/pages/<page-id>/page.manifest.json`

Required top-level fields:
- `id`
- `version`
- `route`
- `title`
- `widgets`
- `filters`
- `actions`
- `permissions`
- `sitemap_node_id`
- `owner`
- `validation`
